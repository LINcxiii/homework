花卉特征分类系统
本项目整体工程结构参考百度飞桨图像分类项目结构搭建，完整复刻数据处理、模型训练、参数存储、样本推理整套工程流程。

一、运行环境
1. 操作系统：Windows10、Windows11均可正常运行
2. Python版本：Python3.6及以上版本，项目基于Python3.11.0完成调试
3. 依赖说明：仅使用python内置json、random、time标准库，无需额外安装第三方包，部署简单便捷
4. 启动方式：进入项目所在文件夹，在文件夹顶部地址栏输入cmd回车打开终端，输入 python main.py 即可启动程序

二、整体功能介绍
1. 数据集构建模块
参考飞桨Dataset设计思路，自动生成多组样本特征数据，完成数据集的整体构建，完成后续训练的数据准备工作。
2. 模型训练与参数存取模块
完成数据集拟合训练工作，支持将训练完成的模型参数保存为本地json文件，同时可以随时读取本地保存的参数完成模型重载，方便重复使用训练好的模型。
3. 样本特征推理分类模块
输入一组样本特征，训练完成后的模型自动完成计算，输出对应的花卉品类，完成样本的分类识别。
4. 推理记录管理模块
程序会自动保存每一次的分类结果与对应时间信息，支持查看全部历史记录，同时可以一键将所有记录保存到本地文本文件中长久留存。

三、运行步骤
1. 将项目内全部文件放置在英文命名的独立文件夹内；
2. 打开该文件夹，在顶部地址栏输入cmd按下回车，唤起命令行终端；
3. 在终端内输入 python main.py 回车，即可启动整套系统；
4. 先输入训练模型完成整体训练，之后可依次使用样本推理、保存模型、加载模型、查看记录、保存记录等功能。

四、补充说明
1. 项目运行后生成的model.json为模型参数存储文件，infer_log.txt为推理记录文件，请勿随意删除修改；
2. 项目整体采用面向对象分层式编写，结构清晰，便于后续进行功能扩充与逻辑修改；
3. 项目全程为独立编写完成，整体架构参考成熟的深度学习工程思路设计。
